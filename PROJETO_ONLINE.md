## 🔄 Como Funciona o Deploy Automático

```
Fazer alterações no código
        ↓
git add .
        ↓
git commit -m "Descrição"
        ↓
git push origin main
        ↓
Streamlit detecta e faz redeploy automático! 🚀
```

**Demora:** ~1-2 minutos para atualizar online

## 📊 Funcionalidades do App

✅ **Análise por Município** - IQESC e todos os indicadores  
✅ **Cálculos Financeiros** - Repasse ICMS Educacional  
✅ **Rankings** - Ordenação por qualquer indicador  
✅ **Estatísticas Gerais** - Visão panorâmica de SC  
✅ **Exportação** - CSV e JSON  
✅ **Relatórios** - Download em TXT  

**Dados:** 288 municípios de Santa Catarina  
**Fonte:** TCE-SC (Tribunal de Contas do Estado)
